## Features

- Tabbed interface: Switch between "Ask AI" and "Search" modes
- Keyboard shortcut: `Ctrl+T` toggles between tabs
- Animated transitions for tabs and input area
- Key focus on accessibility 
  - No `tab` highjack
  - use `CTRL + T` to toggle
  - Proper semantic tags
  - Scope for proper aria-labels
  - No `contentEditable` to prevent script injection risks
- Interruptible animations
